# ORVIX Manifesto

## What ORVIX is not

- **Not a CRM.** We do not optimize for contact records, pipeline stages, or sales rep workflows as the primary model.
- **Not a simple integrator.** We are not a Zapier-style connector list with opaque automation boxes.
- **Not a dashboard template.** We do not ship colorful metric grids, decorative gradients, or feature density for its own sake.

## What ORVIX is

**ORVIX is a decision layer — an operations platform for business events.**

Every business runs on events: a lead arrives, a form fails, a webhook retries, a campaign underperforms, a delivery breaks. ORVIX exists so teams can **see**, **understand**, **decide**, and **act** on those events — with full history and explainability.

## The core cycle

```
Connect → Understand → Decide → Deliver → Observe
```

| Phase | Meaning |
|-------|---------|
| **Connect** | Link sources (Meta, Telegram, future channels) with clear health status |
| **Understand** | Normalize events into a coherent model the team can reason about |
| **Decide** | Apply rules, routing, and human judgment with visible logic |
| **Deliver** | Send outcomes to destinations reliably, with retries and audit trail |
| **Observe** | Measure health, volume, failures, and trends over time |

Today this cycle is expressed as Meta Lead Ads → Telegram. Tomorrow it generalizes to arbitrary sources, pipelines, and destinations.

## Product character

ORVIX must feel:

- **Calm** — no visual noise, no anxiety-inducing reds everywhere
- **Fast** — responsive UI, honest loading states, no blocking mystery
- **Minimal** — every element earns its place
- **Clear** — a new user understands system state within seconds

## Non-negotiables

1. **Every event has history.** Nothing important happens without a trace.
2. **Every decision is explainable.** Users and operators can answer "why did this happen?"
3. **No features for features' sake.** If it does not serve Connect → Observe, it waits.
4. **Quality over breadth.** One module done exceptionally beats five modules half-done.

## For builders

When in doubt, ask:

> Does this help someone understand what is happening and make a better next move?

If the answer is no, do not ship it.
